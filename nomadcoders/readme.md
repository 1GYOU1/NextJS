## Chapter 0 - INTRODUCTION

작업환경 세팅, 프로젝트 생성

@latest는 최신버전이라는 설정
>$ npx create-next-app@latest

typescript를 사용하고 싶다면 뒤에 --typescript 추가
>$ npx create-next-app@latest --typescript

what is your project named? 프로젝트명 입력
>$ nextjs-intro

>$ cd nextjs-intro

>$ npm run dev
